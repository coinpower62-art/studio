# **App Name**: CoinPower

## Core Features:

- User Registration & Authentication: Secure sign-up, login, and session management for users to create and access their accounts.
- Multilingual UI Selection: Enable users to select their preferred language for the application interface from a predefined list.
- Country & Profile Setup: Allow users to select their country (including Italy) from a list of 20 options, accommodating various country codes (e.g., '+1') for registration forms and profile settings.
- Personalized User Dashboard: A customized dashboard displaying an overview of user's investment portfolio, performance summaries, and recent activity.
- Dynamic Company Logo Display: Integrate and prominently display the company logo across various sections of the application for brand identity and consistency.
- AI-powered Market Snapshots: An AI tool providing brief, easy-to-understand summaries of general market trends or notable investment opportunities, personalized based on basic user preferences.

## Style Guidelines:

- Primary color: A radiant gold (#FFD066) chosen to symbolize wealth, achievement, and a premium investment experience.
- Background color: A very soft, off-white hue with a subtle golden undertone (#FCF9F4), providing a clean, clear, and warm canvas for information.
- Accent color: A bold, inviting orange-red (#FB573A) used to highlight interactive elements, calls-to-action, and critical notifications, forming an analogous complement to the primary gold.
- Supplementary color: A fresh, deep green (#4EBB59) employed specifically to represent growth, positive financial indicators, and overall well-being, effectively complementing the gold-based palette.
- Font: 'Inter' (sans-serif), chosen for its modern, clean, and objective aesthetic, making it highly readable and professional for both headlines and detailed financial data.
- Use sleek, professional, and easily understandable icons that clearly represent financial concepts and user actions, subtly incorporating brand colors where appropriate.
- Employ a responsive, organized, and data-focused layout with clear visual hierarchy, particularly for the user dashboard, to present investment information efficiently across all devices.
- Implement subtle and purposeful animations for smooth transitions and visual feedback on interactions, enhancing the user experience without distraction or delays.
